# AI Chatbot with Image Analysis 🤖🖼️

An AI-powered chatbot web application that can:
- Answer text questions
- Analyze uploaded images
- Provide intelligent responses using AI models

## Tech Stack

Frontend:
- React.js
- CSS

Backend:
- Node.js
- Express.js
- Multer (image upload)

AI API:
- Groq API (Llama model)

---

## Features

- Chat interface similar to ChatGPT
- Image upload and analysis
- Fast AI responses
- Full-stack implementation
- Clean UI design
- API-based AI integration
